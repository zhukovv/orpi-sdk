This is the control software for bq24295: 
I2C Controlled 3A Single Cell USB Charger With Narrow VDC 4.5-5.5V Adjustable Voltage at 1.5A Synchronous Boost Operation

After first succesfull run the configuration file with default values automatically will be created in current directory.
Configuration file contains all bits that may be configured. 
If file already exists - the values from it will be used for writing to registers.
